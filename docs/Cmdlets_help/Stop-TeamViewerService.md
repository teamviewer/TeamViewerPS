# Stop-TeamViewerService

## SYNOPSIS

Stop the locally installed TeamViewer service.

## SYNTAX

```powershell
Stop-TeamViewerService [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Stop the locally installed TeamViewer service.
Nothing will happen if TeamViewer is not installed.

## EXAMPLES

### Example 1

```powershell
PS /> Stop-TeamViewerService
```
